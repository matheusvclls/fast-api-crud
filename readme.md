## Installation <a name="installation"></a>

All the libraries required for running the app are listed in the requirement.txt file.
For installing these libraries, only run this comand with Python 3 in root path:

```
$ pip install -r requirements.txt
```


## Instruction <a name="instruction"></a>

To run the API, you only have to run the following command in the root path: 

	```
	uvicorn api:app
	```


## Project Motivation<a name="motivation"></a>

This is a beginner guide to deploy your first API with FlastAPI. Here, we build the CRUD (Create, Read, Update and Delete). To practice, we have a coffee shop menu to use the CRUD methods.
